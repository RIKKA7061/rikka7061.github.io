13자리수 abcdefghijklm 이라 할시
(a + 3b + c + 3d + e + 3f + g + 3h + i + 3j + k + 3l + m) % 10 = 0이다.

저 숫자들 abcde...l 중에 하나를 몰라도
for문으로 1 부터 9까지 하나씩 넣어보면서 10으로 나누어 떨어질 때 그 수가 답이다.

예를 들어, 111111111*113 일 때 *에 들어갈 수는
(1 + (1 x 3) + 1 + (1 x 3) + 1 + (1 x 3) + 1 + (1 x 3) + 1 + (* x 3) + 1 + (1 x 3) + 3) )% 10 = 0 이되야 된다.
즉, 24 + (* x 3) = 30 이므로 *은 2이다.

이제 이 알고리즘을 바탕으로 코드를 짜보자
C#

using System;

class Program
{
    static void Main()
    {
        string input = Console.ReadLine();
        int AsteriskIndex = input.IndexOf('*');
        int hap = 0;
        for (int i = 0; i < 13; i++)
        {
            if (i == AsteriskIndex)
                continue;

            int num = input[i] - '0';
            hap += (i % 2 == 0) ? num : 3 * num;
        }

        for (int x = 0; x <= 9; x++)
        {
            int total = hap + ((AsteriskIndex % 2 == 0) ? x : 3 * x);
            if ((total % 10) == 0)
            {
                Console.WriteLine(x);
                return;
            }
        }
    }
}
